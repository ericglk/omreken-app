# VibeCode Converter

Een kleine Next.js app om getallen om te rekenen tussen decimaal, binair en hexadecimaal.

## Functies

- Decimaal, binair en hexadecimaal omrekenen.
- Hexadecimale invoer wordt automatisch als hoofdletters getoond.
- Oefenmodus voor omrekenen naar binair of hexadecimaal.
- Random oefengetallen van 0 tot en met 255.
- Donkere en lichte modus.

## Starten

Installeer eerst de dependencies en start daarna de ontwikkelserver:

```bash
npm install
npm run dev
```

Open daarna [http://localhost:3000](http://localhost:3000) in je browser.
